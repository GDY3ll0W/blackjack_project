# 🃏 Full-Stack Blackjack Project Setup

This project is a Flutter (Frontend) and Node.js (Backend) application containerized with Docker.

## 🚀 Quick Start (Local Development)

### Prerequisites
* [Docker Desktop](https://www.docker.com/products/docker-desktop/) installed.
* [Flutter SDK](https://docs.flutter.dev/get-started/install) (if running outside of Docker).

### Running the Environment
1. Clone the repository.
2. Open a terminal in the project root.
3. Run the following command:
   ```bash
   docker-compose up --build
